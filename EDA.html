Análise Exploratória de Dados : Base de Comunicação de Acidente de trabalho (CAT) de janeiro a março de 2020¶

by Gilson Castro

Descrição do problema¶

Um labortrio de inovao em segurana ocupacional de uma grande empresa pretende fazer um estudo sobre os acidentes de trabalho no Brasil tendo como objetivo desenvolver servios e produtos para amenizar o problema . Para realizar a anlise , contratou um analista de dados ( especialista em economia do trabalho , este mesmo decidiu realizar uma anlise explortria de dados com o objetivo de obter alguns insights , testar algumas hipteses em torno da base de dados fornecida pelo labortorio.

O analista de dados em uma reunio com o time de pesquisadores elaborou 8 perguntas para responder atravs da anlise explorria:

1) Qual a quantidade de pessoas que recorreu o CAT por sexo ?

2) Qual a quantidade de responsvel ( empregador, sindicato, mdico, segurado, autoridade pblica) pela emisso da CAT ?

3) Quais so as 5 maiores partes do corpo atingida ?

4) Qual o tipo de acidente mais comum ?

5) Quais as 5 maiores classificao Brasileira de Ocupaes apresenta um nmero elevado de acidentes ?

6) Quais leses so mais frequentes ?

8) Onde esto os empregadores ?

Base de dados :¶

O Conjunto de dados escolhido advindo de um servio chamado Registros Comunicao de Acidente de Trabalho - CAT , nesse sistema o trabalhador , empregador registra uma acidente de trabalho ou trajeto . Esse servio tem uma certa facilidade para quem sofre um acidente pois no precisa comparecer na agncia do INSS.
